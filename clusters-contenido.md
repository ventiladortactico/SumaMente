## Estrategia de Enlazado

### Desde Pilar
- Enlace a todas las sub-páginas del cluster
- Enlace a clusters relacionados (ej: Bitcoin → Ethereum, Dólar)

### Desde Sub-páginas
- Enlace al Pilar del cluster
- Enlace a 2-3 sub-páginas relacionadas del mismo cluster
- Enlace a 1-2 sub-páginas de clusters relacionados
- Enlace a herramientas relevantes

### Cross-Linking entre Clusters
- Bitcoin → Dólar (comprar BTC con ARS)
- Plazo Fijo → Inflación (rentabilidad real)
- CEDEARs → Dólar (protección dolarizada)
- Inversión → Educación financiera (aprender antes de invertir)

## Orden de Prioridad

### Fase 1 (Mes 1-2)
1. Cluster Bitcoin (12 artículos + 1 herramienta)
2. Cluster Plazo Fijo (10 artículos + 2 herramientas)

### Fase 2 (Mes 3-4)
3. Cluster Dólar (12 artículos + 3 herramientas)
4. Cluster CEDEARs (6 artículos)

### Fase 3 (Mes 5-6)
5. Cluster Inversión Principiantes (8 artículos + 1 herramienta)
6. Cluster Educación Financiera (10 artículos + 1 herramienta)

## Métricas de Éxito

- Cada pilar debe tener al menos 10 sub-páginas enlazadas
- Cada sub-página debe enlazar al pilar y 2+ sub-páginas relacionadas
- No páginas huérfanas (todas deben tener al menos 3 enlaces entrantes)
- Tiempo promedio en página > 2 minutos
- Tasa de rebote < 60%
